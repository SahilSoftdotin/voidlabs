## 📝 Important Notes

1. **Never commit these folders**:
   - `.next/` - Build cache (auto-generated)
   - `out/` - Static export (auto-generated)
   - `node_modules/` - Dependencies (install via npm)

2. **Always commit these files**:
   - `src/**` - Your source code
   - `public/**` - Static assets
   - Configuration files (next.config.js, etc.)
   - Documentation (README.md, etc.)

3. **Environment-based Configuration**:
   - Local: `basePath: ''`
   - Production: `basePath: '/voidlabs'`
   - Auto-switches based on `NODE_ENV`

4. **Relative Paths**:
   - All internal links use relative paths
   - No hardcoded `/voidlabs` prefix
   - Works on any domain automatically
